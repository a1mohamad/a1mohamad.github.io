CONCLUSIONS

Key Findings

The Linear Regression model provided a stable baseline with an MSE of 0.548. However, the residual analysis reveals that the model struggles with higher-value homes (visible in the outlier plot), likely due to non-linear features like geographic location (Latitude/Longitude) which might require more complex algorithms like Random Forests or XGBoost in future iterations.
